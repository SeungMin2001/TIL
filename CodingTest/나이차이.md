# 문제설명 : 
각각의 나이들의 데이터가 들어오고 그 나이들 사이에서 가장 큰값과 가장 작은값의 차이를 출력하는 문제
```cpp
#include<stdio.h>
#include<algorithm>
using namespace std;
int main(){
	int N,A[101]={0,};
	scanf("%d",&N);
	for(int i=0;i<N;i++){
		scanf("%d",&A[i]);
	}
	sort(A,A+N);
	printf("%d",A[N-1]-A[0]);
	return 0;
}
```
### 풀면서 느낀점 : STL sort를 사용해서 쉽게 풀었던것 같다. 만약 sort를 사용하지 않고 푼다면 반복문을 돌려서 max,min을 if문으로 다 구한다음 차이를 내면 될것같다.
