# Be 동사
>#### 주어가 '~이다' 혹은 '~에 있다' 라는 의미로 사용
>    > 문장 : 주로 명사나 대명사로 시작<br>
>    > 주어 : 문장은 명사나 대명사로 시작하며, 이를 <b>주어</b> 라고 한다<br>
>    > 동사 : 주어에 뒤에서 <b>주어의 상태</b> 나 <b>행동</b> 행동을 설명하여, 동사가 없으면 문장이 성립되지 않습니다. 동사는 크게 <b>be동사</b> 와 <b>일반동사</b> 로 나위어집니다.<br>

<br><br>

>##### 과거시제
> >am -> was<br>
> >is -> was<br>
> >are -> were<br><br>

# There + be동사
#### 어디에 무엇이 있다. (셀수없는 명사 -> 단수)<br><br>

# 일반동사
>#### 일반동사 : 주어의 행동을 설명, be동사를 제외한 모든 동사
>#### 문장당 반드시 1개의 동사가 필요하며, 없거나 여러개를 쓰면 안됨
>#### 주어가 3인칭일떄는 동사의 뒤에 -s,-es,-ies가 붙는다
>#### 3인칭 : 나(I),너(YOU),우리(WE),그들(THEY)을 제외한 나머지가 3인칭 (HE,SHE,IT)<br><br>

# 조동사
>#### be동사나 일반동사의 앞에서 동사의 의미를 더해준다.
>#### 조동사 뒤에는 동사원형 이 온다.<br><br><br><br>


>#### CAN + 동사원형
> >##### ~할수 있다.
> >##### could -> 과거<br><br>

>#### be able to + 동사원형
> >##### can과 의미가 동일하며, 주로 과거,미래 시제와 같이사용.
> >##### ex): will be able to<br><br>

>#### have to + 동사원형
> >##### ~을 해야한다 =>(부정문) :~할 필요가 없다, ~하지 않아도 된다.<br><br>


>#### should + 동사원형
> >##### should는 have to 보다 더 가벼운 정도의 의무나 충고를 나타냄<br><br>

###  *TIP*(표현의 강도!) => need to < sshould < have to < must<br><br><br>

# 문장의 시제
>#### 현재시제 (주어의 현재 상태)
> >##### 확정된 미래의 일정이나 계획에도 현제 시제를 사용가능
> > >###### The meeting starts at 2P.M (회의가 2시에 시작됩니다.)<br><br>
>#### 현재 진행형 시제 (be동사 + 동사ing 패턴)
> >##### 가까운 미래에 계획은 현재 진행형으로 쓸수 있다.
> > >###### We are going there tomorrow (우리는 내일 그곳에 갈것입니다.)<br><br>
>#### 과거시제(주어진 인칭과 상관없이 동사의 과거형을 사용)
> >##### 많은 분들이 답변중에 과거시제를 현재시제로 말하는 실수를 하십니다.
> > >###### 자주 사용ㅎ되는 동사의 과거형, 과거분사형을 꼭 암기하자!<br><br><br><br>


# 현재완료
>#### 현재완료는 과거에 일어난 일이 현재까지 영향을 줄때 사용
>#### 현재완료의 기본 형태는 have(has) + P.P(과거분사) 입니다.
> >##### 현재완료의 부정문은 have뒤에 not,never를 붙힌다<br><br>

>#### 현재완료의 의미
> >##### 현재완료는 계속, 경험의 의미로 사용됩니다.
> > >###### 현재완료 시제는 시간표현 for(~동안), since(~이래로) 와 함께 쓰인다.
> > >###### 현재완료시제는 과거의 시점을 나타내는 표현과 함께 사용되지 않는다.
> > >###### ex) :yesterday, two weeks ago 등등<br><br>

>#### 현재완료의 진행형
> >##### 현재완료의 진행형은 현재완료와 현재진행형이 결합된 시제로 have been + 동사ing 형태로 사용됩니다. 단순 현재완료 시제에 비해 동작의 연속성이 강조됨
> > >###### 현재완료+현재진행형=현재완료 진행<br><br><br>


# To 부정사
>#### to 뒤에 동사원형을 붙여 쓰는것을 to부정사 라고 한다.
>#### 문장에 동사가 더 필요한 경우 to부정사를 사용한다.<br><br>

>#### to 부정사의 명사적 사용
> >##### to부정사는 명사처럼 문장 내에서 주어와 목적어의 역할을 할수 있다.
> > >###### {주어역할} : to부정사가 주어역할을 할때, 문장의 주어 자리에 가주어 It 을 사용 (it 의미없음)
> > >###### {목적어 역할} : to부정사는 특정 동사 다음에 위치해서 목적어 역할을 할수 있다.
> > > >###### to부정사가 목적어 역할을 하는 대표적 동사 : what,decide,need,learn,hope,plan,start,like<br><br>

>#### to부정사의 부사적 사용
> >##### to부정사가 부사의 역할을 하여 동사나 형용사를 꾸밀수 있으며 아래의 세가지 의미를 갖는다.
> > >###### 행동의 목적
> > >###### 감정의 원인
> > >###### 형용사 보강설명<br><br>

>#### For 목적격 + to부정사
> >##### to부정사 앞에 for + 목적격을 더해서 대상을 설명, to부정사의 의미상의 주어 라고도 불림<br><br><br>


# 수동태
>#### 주어가 스스로 어떤 동작을 하는것을 능동태 라고 한다.
>#### 동사를 be + 과거분사의 형태로 바꾸면 주어가 동작을 {받다,당하다} 라는 의미를 가지게 된다. 이를 수동태라고 불른다.<br><br>

>#### 수동태의 기본구조
> >##### 수동태의 부정문 : be동사 + not + 과거분사
> >##### 조동사 + 수동태 : 조동사 + be동사 + 과거분사<br><br>

>#### 수동태의 시제
> >##### 과거시제 + 수동태
> >##### 현재완료 + 수종태
> > >###### ex) The meeting has been cancelled (회의가 취소되었습니다.)
> > >###### 회의 취소의 여파가 현재까지 남아있다면 현재완료 시제를 쓰는것이 좋습니다.
